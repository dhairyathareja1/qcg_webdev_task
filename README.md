# QCG WebD Task

Website submitted to apply as a web developer in QCG.

## Features Included in the Webpage

## Dark / Light Mode

dark mode - light mode toggle switch with seamless functionality

## Club Location

location of the club's office displayed on map

## Social Media Handles

social handles of the club in footer

## Responsive Navigation Bar

responsive navbar which redirects user as per their use-case

## “Apply Now” Page (no libraries used)

a) asks user for details and has checks in place for invalid data (do try entering something which is not an email id)

b) allows user to choose one of the domains

c) generates a quiz from a predefined set of questions

d) evaluates and displays the result of user
